# 산학환류 콘솔 (DX-SPARK)

> 부산과학기술대학교 **산학협력 지식환류 모델** · ANCHOR(DX-SPARK 2040) 사업 통합 관리 플랫폼

기업수요부터 학과매칭 · 과제화 · 현장실습 · 취업 · 성과관리까지 산학협력 전 과정을 **하나의 순환 데이터(환류 루프)** 로 연결하고, **sLLM**이 전 과정을 지능화하는 통합 프로토타입입니다.

> _"흩어진 산학협력 활동을 누적되는 데이터 자산으로 바꾸고, 리포트를 자동 생성한다."_

---

## 📌 개요

| 항목 | 내용 |
|---|---|
| 사업 | 부산과학기술대학교 DX-SPARK 2040 (RISE 단위과제 6) |
| 형태 | 단일 HTML 프로토타입 (React 18 + Babel, 빌드리스) |
| 사용자 역할 | 관리자 · 산학협력단 · 학과담당자 · 기업담당자 · 학생 (5종) |
| 메뉴 | 30+ 화면 · 역할별 권한 분리 |
| 데이터 | 기업·수요·매칭·과제·프로그램·취업·성과 (목업 데이터 + localStorage 영속) |

```bash
open "부산과학기술대 산학환류 콘솔 (오프라인).html"
```

## 🧭 핵심 구조

```
기업수요 → 학과매칭 → 과제화 AI → 프로그램 운영 → 피드백 환류 → 후속수요 ↺
                                            │
   학사(학생행복모델) ┐                      │
   산학협력 운영       ├→ sLLM 종합 엔진 →  ANCHOR 지표 · 환류 리포트
   졸업자 취업현황    ┘
```

- **순환 환류** — 6단계 협력 고리가 끝나면 데이터로 축적되어 다음 매칭의 근거가 됩니다.
- **데이터 통합** — 학사 · 산학협력 · 취업 3개 데이터원이 ANCHOR로 모입니다.
- **sLLM 지능화** — 수요 분석 · 매칭 · 과제화 · 문서작성 · 의사결정 지원을 보조합니다.

## 🗂️ 주요 기능

### 산학협력 환류 코어
- **기업수요** 수집 · 분류 · 상태추적 (등록 → 분석중 → 학과검토 → 과제화)
- **수요-학과 매칭** — AI 적합 학과 추천(점수·사유) · 난이도 · 추천 프로그램 · 유사 사례
- **과제화 AI** — 4단계 위저드로 PoC · 캡스톤 · 재직자교육 후보 생성 + 사업계획서 초안 자동작성
- **LIVING-LAB** — 기술애로 접수 · 컨설팅 · 기술지원 · R&BD 상담 · 후속수요
- **프로그램 운영 / 피드백 환류**

### 현장실습 전주기 관리
수요조사 → 섭외·협약 → 학생확정 → 보험 → 실습진행 → 종합평가 → 성적이관 → 취업연계 (8단계)

### 산학협력 세부활동
산업체특강 · 견학 · 기술지도 · 공동연구 · **연구장비활용** · 캡스톤 · 국책사업 + 전자결재 승인 흐름

### 성과 · 리포트 (ANCHOR)
- **ANCHOR 사업지표** — RISE 7개 단위과제. 과제6 운영데이터 **자동 산정**, 취업현황 **연계 산출**, 그 외 **수기 입력**
- **산학협력 성과·환류 분석** — RISE 차트 · 리드타임 · 재참여 · 수요-역량 갭
- **ANCHOR 환류 리포트** — 학사·산학·취업 데이터를 sLLM이 종합해 통합 리포트 자동 생성

### 졸업자 취업현황
졸업자 1,499명 · 취업률 47.8% · 학과별/업종별/지역정착/성별 인사이트 · 학생 취업상태 실시간 편집

### sLLM · AI 서비스
- **산학 AI 어시스턴트** — 산학문서 자동작성 8종 + 환류 인사이트 Q&A
- **산업도메인 / 전공심화 sLLM 탐색** — 온톨로지 지식맵 · 추천 액션
- **메뉴별 AI 도우미** — 우측 도킹 패널이 페이지마다 역할·추천질문 자동 전환

### 학생 서비스
- **AI 기업·직무 추천** — 전공·역량·포트폴리오 기반 기업·직무 매칭, 부족역량 진단, 현장실습·캡스톤 연결

## 🛠️ 기술 스택

- **Frontend** — React 18.3 (UMD) + Babel Standalone (빌드리스 JSX)
- **상태/저장** — React state + `localStorage` (취업현황·상담기록·소식·테마 영속)
- **스타일** — 단일 `<style>` 디자인 토큰 시스템 (라이트/다크 테마, oklch/color-mix)
- **폰트** — Pretendard
- **배포** — 단일 HTML 인라인 번들(이미지·폰트·PDF·스크립트 포함, 오프라인 동작)
